# omii
omi + i

omii更多关注 esmodule的项目开发。

支持微前端，实现模块动态加载，增量开发。


# getting start

## install
```bash
npm i omii-cli -g
```
## create app
```
omii create my-app
cd ./my-app
npm i
npm start
```
# Target
增加一些OMI支持和关注，构建一个allinone的开发环境！
